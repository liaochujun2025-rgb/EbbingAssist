# UI / UX 样式指南（基于现有 Vue3 Admin 模板）

## 1. 设计方向
- 气质：清爽、理性、学习感；避免花哨渐变，保持干净对齐。
- 布局：保留 Admin 顶栏/侧栏；主内容采用 12 栅格卡片化；仪表盘 2-3 列自适应（xl 3 列，md 2 列，sm 1 列）。
- 组件：Element Plus 优先；卡片 + 表格 + 表单抽屉/对话框；轻阴影、圆角 8px。

## 2. 设计令牌（CSS Variables）
```css
:root {
  --color-bg: #f5f7fb;
  --color-surface: #ffffff;
  --color-primary: #2f80ed;   /* 行动按钮、主图表线 */
  --color-primary-weak: #e8f1fd;
  --color-success: #27ae60;
  --color-warning: #f2c94c;
  --color-danger: #eb5757;
  --color-text: #1f2933;
  --color-muted: #6b7280;
  --radius-md: 8px;
  --radius-lg: 12px;
  --shadow-card: 0 8px 20px rgba(0,0,0,0.04);
  --spacing: 16px;
}
```
Element Plus 主题建议：`primary=#2f80ed`, `success=#27ae60`, `warning=#f2c94c`, `danger=#eb5757`。

## 3. 版式与组件
- 字体：中英混排可用「Inter + 思源黑体」；字号 14/16/20；行高 1.5。
- 按钮：主操作使用实心主色；次操作描边或文本按钮；危险操作红色。
- 表单：分组折行，右对齐标签；关键字段置顶；对话框宽度 520-720px。
- 表格：斑马纹 + 行内操作；状态用彩色标签；工具栏含搜索/筛选/导出。
- 卡片：顶部标题 + 辅助说明；内容保持 16px 内边距；阴影弱化。

## 4. 页面模式
- **仪表盘**：多卡片布局（今日待办、复习待办、番茄计时入口、情绪今日状态、趋势图）。使用折线/柱状组合，卡片右上角含筛选（7/30 天）。
- **学习计划**：列表 + 日历双视图；右上角新增按钮；任务详情用抽屉展示进度/番茄记录。
- **复习中心**：左侧科目/标签筛选，右侧卡片列表；复习打卡面板采用分步（内容 -> 结果按钮 remember/fuzzy/forgot）。
- **情绪日志**：时间轴 + 情绪评分色带；趋势图（折线+面积）；AI 反馈在侧栏卡片展示。
- **设置/偏好**：分区卡片（提醒/语言/个人资料）。

## 5. 交互与状态
- 悬停：按钮/卡片轻微升起（translateY -1px, 阴影增强）。
- Loading：表格/卡片内局部 skeleton；提交按钮 loading 状态。
- 空态：插画 + 简短指引 + CTA（如「去创建计划」）。
- 反馈：删除需二次确认；成功 toast；失败在表单 inline 提示。
- 状态色：
  - 计划/任务：未开始 `#6b7280`，进行中 `#2f80ed`，完成 `#27ae60`，延期 `#eb5757`。
  - 情绪评分：1-5 由红 -> 绿的渐进色条。

## 6. 图表规范
- 使用 ECharts（Admin 模板常用）；主色线条 primary，辅助线 muted。
- 轴/网格简化，留白充足；tooltip 提供日期与完成率/情绪值。
- 圆角柱状图，折线节点小圆点。

## 7. 无障碍与响应式
- 触控友好：按钮高度 ≥ 36px。
- 对比度 ≥ 4.5；文字与图标大小可配置。
- 响应式断点：≥1200px 三列；992-1199 两列；<992 单列堆叠，表格切换为卡片式。
