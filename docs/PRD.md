# Ebbing Assist 产品需求文档（PRD v1.1）

> 版本规划调整：当前交付 **MVP 仅聚焦基础账号体系与个人知识库管理**。艾宾浩斯复习、情绪日志、AI 教练、数据面板等能力保留扩展位，移至 V1.1+。

## 1. 产品概述
- **定位**：面向学习者的个人知识库与基础学习记录工具，基于 Vue3 Admin 模板 + Flask 后端。
- **MVP 目标**：提供可登录的个人空间，支持知识条目的创建/编辑/删除/列表/搜索，并记录当天学习过的条目。
- **扩展方向（V1.1+）**：艾宾浩斯复习系统、情绪日志、AI 教练、数据面板、通知提醒等。

## 2. 用户与场景
- **核心用户**：高校生、职场进阶者、备考人群。
- **典型场景**
  - 整理个人知识：按主题/标签管理笔记或卡片，并快速搜索。
  - 碎片学习记录：记录今天学习过的条目，方便复盘。

## 3. 目标与成功指标
- **MVP 核心指标**：登录成功率 ≥ 99%；知识条目创建成功率 ≥ 99%；搜索响应 < 1s；日均学习记录创建率 ≥ 50%（活跃用户）。
- **界面要求**：沿用 Admin 模板的美观度与信息密度，不自创布局。

## 4. 功能范围与优先级
### MVP（P0）
1) **账号与基础**
   - 邮箱/手机号注册登录，JWT 会话；个人资料（昵称、头像、时区）；基本偏好设置（语言占位）。
2) **个人知识库管理**
   - 主题/分类（可选）与标签。
   - 知识条目（笔记/卡片）：标题、正文/内容、标签、所属主题、可选链接或附件占位。
3) **知识 CRUD 与搜索**
   - 新增/编辑/删除/列表；按关键字、标签、主题筛选；分页与排序。
4) **学习记录（简版）**
   - 记录今日学习的条目（关联知识条目 ID + 备注 + 时间）。
   - 查看近 7 天记录列表。

### V1.1+（后续版本，保留扩展位）
- 艾宾浩斯复习：间隔计划生成、打卡、统计。
- 情绪日志与 AI 教练：情绪记录、趋势、AI 文本反馈。
- 数据面板：今日待办/复习、学习趋势、搜索热词。
- 通知与提醒：到期提醒（站内/邮件占位）。
- 模板中心、协作/共享、导入导出、移动端/PWA。

## 5. 用户故事（MVP）
- 作为用户，我可以注册/登录并进入个人空间。
- 作为学习者，我可以创建带标签/主题的知识条目，并随时编辑或删除。
- 作为学习者，我可以按关键词或标签快速搜索知识条目。
- 作为学习者，我可以记录今天学了哪些条目，并查看最近几天的记录。

## 6. 业务流程（MVP）
- **登录**：注册/登录 -> JWT 下发 -> 进入知识库首页。
- **知识条目管理**：新建/编辑表单 -> 保存 -> 列表/搜索展示。
- **学习记录**：选择已学条目 + 备注 -> 保存 -> 近期列表展示。

## 7. 信息架构 / 模块
- 账号/认证
- 知识库：主题/标签、条目列表/详情/搜索
- 学习记录：记录创建、近 7 天列表
- 设置：个人资料、语言占位
- （预留）复习中心、情绪日志、AI 教练、仪表盘、通知

## 8. 数据要素（MVP）
- 用户：id，email/phone，password_hash，nickname，avatar，timezone，prefs(json)。
- 主题/分类（可选）：id，user_id，name，desc。
- 知识条目：id，user_id，title，content，tags(text[])，topic_id，links(json)，created_at，updated_at。
- 学习记录：id，user_id，entry_id，note，logged_at。
- （预留）复习计划/打卡、情绪日志、AI 反馈、通知等表在 V1.1+ 扩展。

## 9. 非功能需求
- 性能：列表/搜索首屏 < 1s（小规模数据），写操作 < 400ms。
- 安全：JWT + HTTPS；基础权限校验；输入校验与防注入。
- 国际化：保留 i18n 占位（中/英）。

## 10. 验收标准（MVP）
- 能注册/登录并查看个人资料。
- 能创建/编辑/删除知识条目，并按标签/关键词搜索。
- 能创建学习记录并查看最近 7 天记录。
- UI 复用 Admin 模板，布局整齐，适配主流桌面分辨率。
- 预留扩展位（复习/情绪/AI/通知）但不实现具体功能。

## 11. 风险与对策
- 数据扩展：为复习/AI 等留字段与表命名空间，保持前后端可演进。
- 搜索性能：前期用 DB 简单模糊匹配，后续可接入全文索引。
- 复杂度控制：MVP 不做复习算法与 AI，避免超范围。

## 12. 里程碑（调整后）
- W1：PRD/架构更新，账号 + 知识条目基础 API/页面。
- W2：知识搜索、标签/主题管理、学习记录。
- W3：联调与基本测试，部署候选。
- V1.1+：复习系统、情绪日志、AI 教练、通知、数据面板。
